4. **app/main.py** - FastAPI app:
```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI(
    title="OmniRom API",
    description="Universal Romanization Service - Convert any script to Latin",
    version="0.1.0"
)

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/")
def root():
    return {"message": "OmniRom API", "version": "0.1.0"}

@app.get("/health")
def health():
    return {"status": "healthy"}
```

<name>Task 2: Create Dockerfile</name>
<files>Dockerfile</files>
<action>
Create Dockerfile for Render deployment using uv:
```dockerfile
FROM python:3.11-slim

# Install uv
RUN pip install uv

WORKDIR /app

# Copy pyproject.toml first for better caching
COPY pyproject.toml .

# Install dependencies with uv
RUN uv sync --frozen --no-dev

COPY . .

# Run with uvicorn
CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
```
