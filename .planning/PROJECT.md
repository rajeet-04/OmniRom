# OmniRom Project

## Vision
Build a highly accurate, low-latency "any-to-Latin" transliteration API that dynamically detects input script and routes to the best language-specific engine.

## Tech Stack (Free)
- **Backend:** Python + FastAPI
- **Hosting:** Render (free tier)
- **Caching:** Aiven Valkey (free Redis-compatible)
- **Task Queue:** Celery + Redis
- **Container Registry:** GitHub Container Registry
- **CI/CD:** GitHub Actions

## Constraints
- NO paid resources
- Must work within free tier limits
- Self-hosted NLP engines only (no paid APIs)
